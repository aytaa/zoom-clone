# Zoom Clone 

Zoom Clone maked node js express server.

## Installation

Start server [npm](https://nodejs.org/) to install.

```bash
npm install 
```
```bash
npm run start:dev 
```

## Usage

```node js
import foobar

foobar.pluralize('word') # returns 'words'
```
